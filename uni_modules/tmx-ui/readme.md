# TMUI4.0x

## 设计

一款好的UI/UX，不止是功能，它还应该要有设计感，层次感。注重细微及用户的反馈体验。同时要保留它的高度可扩展性及性能。
功能的堆叠不是我所追求，也不是我的目标。我追求的是细致，可扩展，精美。不管是IOS，安卓，WEB/H5三端我都朝着这个方向。
让UNIAPPX 开发的商业应用更像商业应用。为让市场的组件审美提高一个档次而做出应有的贡献。
